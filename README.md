# Quantum Error Correction Course

Short course materials for learning the basics of quantum computing and quantum
error correction.

## Contents

- `Docs/` - introduction and recommended resources
- `Exercises/` - Jupyter notebooks with explanations and exercises
- `requirements.txt` - Python packages needed to run the notebooks

## Setup

```bash
pip install -r requirements.txt
jupyter notebook
```

Start with `Docs/Introduction_and_resources.md`, then work through the notebooks
in `Exercises/`.
